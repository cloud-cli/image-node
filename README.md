# Node.js

This is a [Node.js](https://nodejs.org/) base image for [Cloudy](https://github.com/cloud-cli/cloudy)

### Installation

```bash
cy install nodejs
```

### Usage

Create a Cloudy service configuration and use the type "nodejs".

```json
{
  "type": "nodejs",
  ...
}
```

Read the Cloudy documentation for more details.
